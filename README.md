# Sudoku Web App | 在线数独网页应用

一个轻量级、功能完备的纯前端数独游戏。支持多种难度选择、实时交互反馈，以及强大的 PDF 题目打印生成功能。

## 🌟 核心特性

- **🧠 五级难度设定**：从“儿童模式（入门）”到“大师级”，满足不同阶段玩家需求。
- **✨ 智能视觉反馈**：
  - **相同数字高亮**：选中任意数字，全盘相同数字同步高亮，辅助逻辑推导。
  - **实时错误检查**：一键校验，错误数字红字标出。
  - **状态区分**：清晰区分题目预置数字（深色加粗）与用户输入数字（蓝色）。
- **📱 响应式设计**：适配 PC 与移动端，提供虚拟数字键盘。
- **🖨️ PDF 题目生成**：
  - 支持批量生成题目。
  - 可选择每页 1 个、2 个或 4 个题目的布局。
  - 完美适配 A4 纸张打印，适合线下练习。
- **🚀 零依赖部署**：纯 HTML/CSS/JS 构建，无需后端，秒速开启。

## 🛠️ 技术栈

- **HTML5**: 结构化内容与打印布局。
- **CSS3**: 响应式布局、Flexbox/Grid 布局及 `@media print` 打印优化。
- **Vanilla JavaScript (ES6+)**: 
  - 核心算法：回溯法 (Backtracking) 生成唯一解及求解。
  - 交互逻辑：面向对象编程 (OOP) 管理游戏状态。

## 🚀 如何使用

### 本地运行
1. 下载或克隆本项目。
2. 直接双击 `index.html` 即可在浏览器中运行。

### 部署建议
- **GitHub Pages**: 将代码推送到仓库后，在 Settings -> Pages 中开启。
- **Vercel / Netlify**: 拖拽文件夹即可完成全球 CDN 部署。

## 🎮 操作说明

1. **选择难度**：在下拉菜单选择合适难度。
2. **填写数字**：
   - 点击格子，使用键盘数字键或下方虚拟按键填入。
   - 点击空白处或非数字区可取消选中并清除高亮。
3. **辅助**：点击任意已有数字，可以快速定位盘面中所有相同的数字。
4. **校验**：点击“检查答案”，系统将标记出错误的数字。
